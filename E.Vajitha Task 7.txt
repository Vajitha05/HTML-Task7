1.Learn the difference between unordered (<ul>) and ordered (<ol>) lists. Create shopping list and steps list.

Ans: <ul> (Unordered L ist)
     Items are shown with bullet points (•)
     Order is not important

     <ol> (Ordered List)
     Items are shown with numbers (1, 2, 3...)
     Order is important (step by step)

<ul> Shopping list

<!DOCTYPE html>
<html>
<head>
    <title>Shopping List</title>
</head>
<body>

<h3>Shopping List</h3>
<ul>
  <li>Rice</li>
  <li>Milk</li>
  <li>Bread</li>
  <li>Eggs</li>
</ul>

</body>
</html>

 <ol> Shopping list

<!DOCTYPE html>                                          
<html>

    <title>Steps List</title>

4.Add headers to tables using <th>. Compare difference between <td> and <th>.

Ans: <th> (Table Header)
     Used for headings
     Text is bold
     Default center aligned
     Usually used in the first row

     <td> (Table Data)
     Used for data (content)
     Text is normal
     Default left aligned
     Used in other rows


<!DOCTYPE html>

5.Practice table attributes: colspan, rowspan, and add borders, cell spacing, and padding with CSS.

Ans:  <!DOCTYPE html>
<html>
<head>
    <title>Table Practice</title>

    <style>
        table {
            border-collapse: collapse;
            width: 60%;
        }

        th, td {
            border: 1px solid black;
            padding: 10px;
            text-align: center;
        }

        table {
            border-spacing: 10px;
        }
    </style>
</head>
<body>

<h3>Student Table</h3>
